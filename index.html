匯入 n8n 教學

      


        

          
Step 1：匯入 Workflow

          
    
            
  1. 開啟 n8n Dashboard
    2. 
            
  2. 左側選單點選 Workflows
    3. 
            
  3. 右上角點擊 ⋮ 選單 → Import from File
    4. 
            
  4. 選擇本專案的 workflow.json
    5. 
            
  5. 確認匯入
    6. 
          

        


        

          
Step 2：建立 Service Account

          
    
            
  1. 在 Google Cloud Console 建立專案並啟用 Google Sheets API
    2. 
            
  2. 進入「IAM & Admin → Service Accounts」→ Create Service Account
    3. 
            
  3. 輸入名稱（例如 n8n-sheets-reader）→ Create → Done
    4. 
            
  4. 點擊剛建立的 Service Account → Keys 分頁 → Add Key → JSON
    5. 
            
  5. 下載 JSON 金鑰檔案，記下 Service Account Email
    6. 
          

        


        

          
Step 3：共用 Sheet + 綁定 Credential

          
    
            
  1. 打開 7 個倉的 Google Sheet → 「共用」→ 加入 Service Account Email 為「檢視者」
    2. 
            
  2. 在 n8n → Settings → Credentials → Add → Google API (Service Account)
    3. 
            
  3. 填入 Service Account Email + Private Key（從 JSON 金鑰檔案複製）
    4. 
            
  4. Scopes 填入 https://www.googleapis.com/auth/spreadsheets.readonly
    5. 
            
  5. 匯入 Workflow 後，點擊「讀取 Google Sheets」節點，確認 Credential 已綁定
    6. 
          

        


        

          
Step 4：啟用與測試

          
    
            
  1. 右上角切換 Workflow 為 Active
    2. 
            
  2. 使用下方 curl 指令測試：
    3. 
          

          
curl -X POST http://localhost:5678/webhook/attendance-query \
  -H "Content-Type: application/json" \
  -d '{"query": "4/2請假狀況"}'

        


        

          
擴充 Email 報表

          
    
            
  1. 在「回傳成功結果」前新增 IF 節點，判斷 needReport === true
    2. 
            
  2. 新增 Send Email 或 Gmail 節點
    3. 
            
  3. 使用 emailSubject 作為主旨
    4. 
            
  4. 使用 emailBody + rows 格式化作為內文
    5. 
            
  5. 如需 Excel 附件，新增 Spreadsheet File 節點轉為 xlsx
    6.
